**The Stone Throwers of Babylon Present**
# From On High

A game of caveman AIs fighting for dominance in a shattered world, while devious gods sceme in the world beyond.\
\
*After a disasterous class war, the world lies in absolute ruin, twisted magics bending the landscape to be nearly impassable. Few survivors remain, and those who do have nearly no technology left to them. Whisper in the ears of these survivors as one of the Ascended, gods of this new world order, and shape the world to your liking.*

## How to Set Up

To run this game, Python 3.10 - 3.12 is required. [Python 3.12](https://www.python.org/downloads/release/python-3120/)\
\
Install pygame with 
```python
pip install pygame
```
After those are installed, the game can be run with 
```python
python3 main.py
```
